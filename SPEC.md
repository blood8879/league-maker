  - [x] 태그 표시

- [x] 필터링 기능
  - [x] 지역별 필터
  - [x] 실력별 필터
  - [x] 모집 상태별 필터
- [x] 정렬 기능
  - [x] 최신순 정렬
  - [x] 인기순 정렬
  - [x] 멤버 수순 정렬

#### `/teams/new` 팀 생성 페이지
- [x] 팀 기본 정보 입력 폼
  - [x] 팀명 입력 (중복 확인 UI)

### ✅ 구현할 기능

#### `/matches/new` 경기 생성 페이지
- [x] **Step 1: 경기 유형 선택**
  - [x] 🏆 리그/컵 경기 (공식 경기) 옵션
  - [x] ⚽ 친선 경기 옵션
  - [x] 🏃 연습 경기 옵션
  - [x] 선택한 유형에 따라 다른 폼 표시
- [x] **Step 2-A: 리그/컵 경기**
  - [x] 리그/컵 선택 드롭다운
  - [x] 홈팀/원정팀 자동 표시 (리그 일정 기반)
  - [x] 등록 선수만 참가 가능 안내 메시지
- [x] **Step 2-B: 친선/연습 경기**
  - [x] 내 팀 선택
  - [x] 상대팀 선택 (검색 또는 "상대 없음" 옵션)
  - [x] 용병 모집 여부 체크박스

  - [x] 경기 날짜/시간 (DateTimePicker)

- [x] **Step 4: 참석 인원 초기 설정**
  - [x] 팀 멤버 목록 표시
  - [x] 각 멤버 옆에 "참석 예상" 체크박스

- [x] 경기 기본 정보 헤더
- [x] 경기 상태 뱃지 (예정/진행중/종료)
- [x] **참석 관리 섹션** (팀 멤버만)
  - [x] 내 참석 여부 버튼 (참석 ✓ / 불참 ✗ / 미정 ?)
  - [x] 참석/불참/미정 인원 수 표시

## Phase 4: 경기 기록 입력 페이지 구현

### 📋 개요
경기 진행 중 실시간으로 경기 기록을 입력할 수 있는 페이지를 구현합니다. **참석한 선수만** 기록에 포함될 수 있도록 합니다.

### ✅ 구현할 기능

#### `/matches/[id]/record` 경기 기록 입력 페이지
- [x] **상단 스코어보드**
  - [x] 홈팀 vs 원정팀 스코어 (실시간 업데이트)
  - [x] 타이머 (전반/후반/종료)
  - [x] 시작/일시정지/재개/종료 버튼
- [x] **선수 선택 모드**
  - [x] **참석한 선수만 표시** (참석 확인된 선수 + 승인된 용병)
  - [x] 탭으로 홈팀/원정팀 전환
  - [x] 선수 카드 (사진, 이름, 포지션, 등번호)
  - [x] 선택된 선수 하이라이트
- [x] **빠른 기록 버튼**
  - [x] ⚽ 득점 (선수 선택 → 어시스트 선택(선택사항) → 시간 자동/수동)
  - [x] 🟨 경고 (선수 선택 → 사유 선택)
  - [x] 🟥 퇴장 (선수 선택 → 사유 선택)
  - [x] 🔄 교체 (OUT 선수 → IN 선수)
- [x] **기록 타임라인**
  - [x] 시간순 이벤트 목록
  - [x] 이벤트 아이콘 + 선수명 + 시간
  - [x] 수정/삭제 버튼 (각 기록)
- [x] **라인업 관리**
  - [x] 선발 11명 선택 (또는 자동 배치)
  - [x] 교체 선수 목록
  - [x] 포메이션 표시 (선택)
- [x] 경기 종료 확인 모달

#### 입력 제한 로직
- [x] 리그/컵: 등록 선수만 선택 가능
- [x] 친선/연습: 참석 확인 + 승인된 용병만 선택 가능
- [x] 불참/미정 선수는 선택 불가 (회색 처리)

#### 공통 작업
- [x] 모바일 최적화 (세로 모드, 큰 터치 영역)
- [x] 가짜 데이터 생성 (각 경기당 10-20개 이벤트)

### 🔍 검증 항목
- [x] 참석한 선수만 선택 목록에 표시되는지 확인
- [x] 불참/미정 선수가 회색 처리 또는 숨김되는지 확인
- [x] 득점 입력 시 스코어보드가 즉시 업데이트되는지 확인
- [x] 어시스트를 선택적으로 입력할 수 있는지 확인
- [x] 교체 입력 시 OUT/IN 선수가 올바르게 구분되는지 확인
- [x] 타임라인에 모든 이벤트가 시간순으로 표시되는지 확인
- [x] 기록 수정/삭제가 즉시 반영되는지 확인
- [x] 경기 종료 시 확인 모달이 표시되는지 확인
- [x] 모바일에서 버튼이 충분히 크고 터치하기 편한지 확인

#### `/matches/[id]` 경기 상세 페이지 (경기 후)
- [x] 경기 결과 헤더 (최종 스코어, 날짜, 장소)
- [x] 탭 네비게이션 (개요, 기록, 라인업, 통계, 참석)
- [x] **개요 탭**
  - [x] 득점 타임라인 (시간대별 득점 표시)
  - [x] 주요 이벤트 요약
  - [x] MVP 선정 (득점왕, 어시스트왕)
- [x] **기록 탭**
  - [x] 전체 이벤트 목록 (득점, 경고, 퇴장, 교체)
  - [x] 팀별 필터링
  - [x] 시간대별 그룹핑
- [x] **라인업 탭**
  - [x] 양팀 선발 라인업 (포메이션)
  - [x] 교체 선수 목록
  - [x] 각 선수 기록 (득점, 어시스트, 카드)
- [x] **통계 탭**
  - [x] 간단한 경기 통계
  - [x] 슈팅, 파울, 코너킥 등 (입력된 경우)

  - [x] 최종 참석 인원
  - [x] 불참 사유 (선택적)
  - [x] 용병 정보 (있는 경우)
